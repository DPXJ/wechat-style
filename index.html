> [!info] 文字信息提示（绿色横幅）
> [!tip] 文字提示框（绿色横幅）
> [!warning] 文字警告框（橙色横幅）
:::info ... :::信息块（整块绿色背景）
:::tip ... :::提示块
:::card ... :::卡片块（深色卡片背景）
:::dark ... :::暗色卡片（更深的背景）
:::summary ... :::总结块 — 浅黄高亮框（镜哥主题专属）
:::digest ... :::阅前须知 — 灰底卡片 + 虚线分段
:::numbered ... :::数字徽章 — 圆形序号 + 无框排列
:::intro ... :::侧边线 — 无外框，每段左侧竖线
:::tail ... :::尾图 — 放在点赞区之后，带外框
:::quote ... :::引用A — 大引号 + 左彩线，末段为来源
:::quote-b ... :::引用B — 渐变背景 + 上下彩线 + 斜体
:::quote-c ... :::引用C — 头像气泡对话风格
:::flow ... :::竖排流程 — 序号圆 + 连线，每段为一步
:::stat ... :::数据卡片 — 每行：数字|标签|说明
:::compare ... :::对比块 — 用 --- 分隔左右列
:::callout 🚀 ... :::醒目强调 — 自定义图标 + 渐变背景
:::highlight ... :::高亮大字 — 双侧色条 + 居中大字
:::divider A/B/C:::装饰分割线：A渐变线 B菱形 C圆点
:::center ... :::居中块
:::subtitle ... :::副标题（带绿色圆点）
:::checklist 标题 ... :::终端/清单块（macOS 窗口风格）
:::terminal 标题 ... :::同上，代码风格
:::grid-2 ... :::2列图片网格
:::grid-3 ... :::3列图片网格
