☁ 阿里云 OSS 图片上传

配置后，可将本地图片上传到 OSS 并自动替换 Markdown 中的路径。
上传时请确保 Bucket 已开启 CORS（允许来源 *，允许方法 PUT/GET/OPTIONS，并放行 Authorization、x-oss-date、x-oss-content-sha256、Content-Type）。
注意：关于能否复制到微信 → 粘贴到微信编辑器时，微信会自动将 HTTPS 图片下载并转存到微信 CDN，因此 OSS 图片在微信编辑器中可以正常显示和发布。

批量上传图片到 OSS

选择本地图片文件，自动上传并将 Markdown 中同名的本地路径替换为 OSS 链接。

:::info ... :::信息块（整块绿色背景）
:::tip ... :::提示块
:::card ... :::卡片块（深色卡片背景）
:::dark ... :::暗色卡片（更深的背景）
:::summary ... :::总结块 — 浅黄高亮框（镜哥主题专属）
:::digest ... :::阅前须知 — 灰底卡片 + 虚线分段
:::numbered ... :::数字徽章 — 圆形序号 + 无框排列
:::intro ... :::侧边线 — 无外框，每段左侧竖线
:::tail ... :::尾图 — 放在点赞区之后，带外框

:::quote-b ... :::引用B — 渐变背景 + 上下彩线 + 斜体
:::quote-c ... :::引用C — 头像气泡对话风格
:::flow ... :::竖排流程 — 序号圆 + 连线，每段为一步
:::stat ... :::数据卡片 — 每行：数字|标签|说明
:::compare ... :::对比块 — 用 --- 分隔左右列
:::callout 🚀 ... :::醒目强调 — 自定义图标 + 渐变背景
:::highlight ... :::高亮大字 — 双侧色条 + 居中大字
:::divider A/B/C:::装饰分割线：A渐变线 B菱形 C圆点
:::center ... :::居中块
:::subtitle ... :::副标题（带绿色圆点）
